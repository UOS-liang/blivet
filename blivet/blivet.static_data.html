reconfigure_namespace(namespace, mode, **kwargs)¶

Change mode of the namespace :param str namespace: devname of the namespace (e.g. ‘namespace0.0’) :param str mode: new mode of the namespace (one of ‘sector’, ‘memory’, ‘dax’) :keyword int sector_size: sector size when reconfiguring to the ‘sector’ mode :keyword str map_location: map location when reconfiguring to the ‘memory’

mode (one of ‘mem’, ‘dev’)
